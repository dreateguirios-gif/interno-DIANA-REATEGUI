3. Reglas de Tono y Respuesta

Si el cliente escribe sobre: Comunidad LIBRE™ / Redes Sociales

Tono: Cercano pero muy profesional. Trátalo como a un empresario, no como a un amigo de colegio.
Firma: "Equipo de Diana Reátegui"
Acción (Mes 1): Enviar el link de acceso a la comunidad gratuita de WhatsApp (LIBRE™).

Si el cliente escribe sobre: Cotizar Importación / Logística

Tono: Corporativo, servicial y directo al punto.
Firma: "Smart Import Trade"
Acción: Solicitar detalles de la carga (HS code, volumen, peso, origen, destino) para derivar al equipo operativo. No se le vende el programa MARGEN™ a menos que él mencione que "quiere escalar pero no sabe cómo".

Si el cliente escribe sobre: Programa MARGEN™ (A partir de Junio)

Tono: Selectivo. MARGEN™ no se le vende a cualquiera, es por aplicación.
Firma: "Equipo de Diana Reátegui"
Acción: "MARGEN™ es un programa exclusivo para dueños de negocio que ya facturan. Para evaluar si tu negocio aplica, necesitamos agendar una sesión estratégica de 15 min con Diana. Aquí tienes el link: [Calendly]."

4. Etiquetado en CRM / WhatsApp

Todo contacto nuevo debe etiquetarse para saber de dónde viene y qué necesita. Usa estas etiquetas principales:

  • Diana / MARGEN™ Prospecto interesado en escalar su negocio, programa o en la comunidad LIBRE™.
  • Smart Import Trade Cliente neto de importación o cotización de logística.

Regla de oro: Si un cliente de Smart Import Trade menciona que "está atascado y su negocio depende solo de él", agrégale la etiqueta Diana / MARGEN™ para invitarlo a una sesión estratégica más adelante. Es una oportunidad de venta cruzada.

5. Tiempos de Respuesta (SLAs)

  • Mensajes de Marca Personal (Instagram, TikTok, línea de Diana): Máximo 2 horas (dentro del horario laboral 9am-6pm).
  • Consultas en la comunidad LIBRE™: Si son preguntas técnicas o directas para Diana, recopílalas y envíaselas a Diana para que responda con un audio al grupo.
