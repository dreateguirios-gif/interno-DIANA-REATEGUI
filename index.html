02 — Cómo usar este sistema

Para empezar
a operar hoy

1
Lee tu documento prioritario

Cada rol tiene un documento marcado como "Prioritario" arriba. Ese es el único que necesitas leer en tu primer día. Es el manual de tu trabajo. Todo lo demás es referencia.

2
Consulta el plan de acción

El Plan de Acción dice qué se hace esta semana y quién lo hace. Es el documento que se actualiza más frecuentemente y donde coordina el equipo.

3
Ante dudas, consulta los SOPs

Los SOPs tienen el procedimiento exacto para cada situación operativa: cómo editar, cómo responder, qué hacer cuando hay algo fuera de lo normal. No adivines — busca el SOP primero.

Regla del equipo: Si algo no está documentado aquí, se lo dice a Diana y se documenta. La operación no depende de la memoria de nadie — depende del sistema.
